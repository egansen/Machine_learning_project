This notebook predicts NBA home team wins by comparing two machine learning models — Random Forest and Logistic Regression — using the games.csv dataset. After cleaning the data and selecting key features (field goal %, free throw %, three-point %, and rebounds), both models are trained on a 70/30 split. Logistic Regression outperforms Random Forest (77% vs. 75% accuracy), a result further confirmed by 5-fold cross-validation at 76%. A statistical analysis using statsmodels shows all features are highly significant, with field goal percentage being the strongest predictor of a home win, followed by rebounds.
